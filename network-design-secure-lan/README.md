# Design a Secure Local Area Network (LAN)

## Project Overview

This project demonstrates the design of a secure Local Area Network (LAN) using standard networking components and security controls such as firewalls, routers, switches, and wireless access points.

The objective of this project was to understand how network devices are positioned within a network to support connectivity while maintaining security through segmentation and controlled access.

---

## Network Components

* Internet
* Firewall
* Router
* Switch
* Server
* Wireless Access Point (WAP)
* End-user devices (workstations, laptops)

---

## Security Design Features

* Network segmentation using firewalls
* Controlled wireless access through a dedicated firewall
* Separation of internal devices from external networks
* Protection of internal resources from unauthorized access

---

## Skills Demonstrated

* Network architecture design
* Secure network segmentation
* Understanding of firewalls and access control
* Cybersecurity fundamentals
* LAN topology planning
