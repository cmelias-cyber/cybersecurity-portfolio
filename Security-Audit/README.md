## Security Audit – Botium Toys

This folder contains my internal security audit for the fictional company Botium Toys, completed as part of the Google Cybersecurity Certificate program.

The audit evaluates the organization’s security controls and compliance practices using the NIST Cybersecurity Framework (CSF). It identifies potential risks, threats, and vulnerabilities that could impact business operations and critical assets.

This exercise demonstrates my ability to assess security posture, review compliance requirements, and apply structured frameworks to real-world scenarios.

The findings and checklist results are documented in the security audit file.
