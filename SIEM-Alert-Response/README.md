# SIEM Alert Response 

## Overview
This folder contains a cybersecurity training exercise completed as part of the Google Cybersecurity Certificate program. The activity focuses on responding to a simulated SIEM alert involving a suspicious file download linked to a potential indicator of compromise (IOC).

## Purpose
The goal of this exercise is to demonstrate foundational skills in SOC operations, including alert triage, incident validation, and structured response workflows.

## Skills Demonstrated
- SIEM alert interpretation  
- Incident triage and validation  
- SOC incident response workflow understanding  
- Threat containment and recovery concepts  
- Security documentation  

## What This Shows to Employers
This artifact demonstrates my ability to understand and follow structured SOC processes when responding to security alerts, including how to assess potential threats and document incident handling procedures.
