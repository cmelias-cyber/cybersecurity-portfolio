# SIEM Alert Response – Practice Activity

## Overview
This folder contains my portfolio artifact for the activity: **Practice: Respond to a SIEM alert** from the Google Cybersecurity Certificate program.

## Purpose
The purpose of this exercise was to practice responding to security alerts generated by a Security Information and Event Management (SIEM) system. The scenario involved investigating a suspicious file download associated with a potential indicator of compromise (IOC), and following a structured incident response process.

## Key Focus Areas
- SIEM alert analysis and validation  
- Incident response workflow (assessment, containment, recovery, communication)  
- Application of incident response playbooks  
- Security operations decision-making  

## What This Demonstrates
This exercise demonstrates my ability to:
- Interpret and respond to security alerts in a SOC environment  
- Follow structured incident response procedures  
- Apply critical thinking when evaluating potential threats  
- Document incident handling steps clearly and professionally  

## Note
This artifact is part of my early cybersecurity portfolio and reflects foundational training in security operations and incident response practices.
