# Secure Local Area Network (LAN) Design

## Overview

This project demonstrates the design of a secure Local Area Network (LAN) using common networking components such as routers, firewalls, switches, a server, and a wireless access point (WAP).

The goal of this design is to show how network devices are organized to support connectivity while maintaining security through segmentation and layered protection.

---

## Key Security Focus

* Network segmentation
* Firewall placement and traffic filtering
* Secure wireless access
* Controlled internal connectivity
* Defense-in-depth principles

---

## Skills Demonstrated

* Network architecture design
* Understanding of LAN components
* Cybersecurity fundamentals
* Secure network planning
* Technical documentation
